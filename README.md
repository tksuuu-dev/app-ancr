# Ancr (macOS)

ブックマーク・フォルダ・ファイルをカテゴリーとホットキーで整理し、すばやく開ける macOS アプリ。

A macOS app to manage and instantly open bookmarks, folders, and files.<br>
Organize with categories, hotkeys, and custom open-with settings.

---

**このリポジトリは配布専用です。**

## インストール方法
1. [Releases](https://github.com/tksuuu-dev/app-ancr/releases) から最新の `.dmg` をダウンロードします。
2. アプリケーションフォルダにコピーします。
3. 初回起動時は **右クリック > 開く** を選択してください。
